# ☕ Coffee Shop Sales Dashboard

## 📌 Overview

This project is a **learning-purpose Excel Dashboard** created to practice data analysis, dashboard design, and business reporting using **Microsoft Excel**.

The dashboard provides an interactive overview of coffee shop sales, allowing users to analyze sales performance by coffee type, country, roast type, order date, and other filters. The main objective of this project was to improve Excel dashboard development skills by working with real-world business data.

## 🎯 Project Objectives

- Practice Excel dashboard creation.
- Learn data cleaning and transformation.
- Build interactive reports using Pivot Tables and Pivot Charts.
- Analyze coffee sales across different countries.
- Create user-friendly filters using Slicers and Timeline.

---

## 📊 Dashboard Features

The dashboard includes:

- **Total Sales Overview**
  - Monthly sales trend visualization.

- **Sales by Country**
  - Compare sales performance across countries.

- **Coffee Type Sales**
  - Total sales for:
    - Robusta
    - Liberica
    - Excelsa
    - Arabica

- **Interactive Filters**
  - Loyalty Card
  - Roast Type
  - Coffee Size
  - Order Date Timeline

- **Dynamic Dashboard**
  - Charts automatically update based on selected filters.
